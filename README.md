# Biblioteca Digital - Projeto Acadêmico

Sistema de gerenciamento de acervo desenvolvido como parte da graduação em Ciência da Computação na **Cruzeiro do Sul**.

## 🚀 Tecnologias Utilizadas
* **Java 21** (JDK)
* **JavaFX 21.0.1** (Interface Gráfica)
* **Maven** (Gerenciador de Dependências)
* **MySQL** (Banco de Dados)
* **Scene Builder** (Design de UI)

## 📋 Funcionalidades em Desenvolvimento
- [x] Interface inicial de Login
- [ ] Cadastro de Usuários/Alunos
- [ ] Consulta de livros no acervo
- [ ] Sistema de empréstimos e devoluções

## ⚙️ Como Executar
1. Certifique-se de ter o **JDK 21** e o **Maven** instalados.
2. Configure o banco de dados localmente (MySQL).
3. Renomeie o arquivo `config.properties.example` (ou crie um novo) para `config.properties` com suas credenciais locais.
4. Execute via terminal:
   ```bash
   mvn javafx:run
